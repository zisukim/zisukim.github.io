Problem

Solution

  • If you solved this problem to $O(n^3)$, It would be failed due to the timeout.
  • You should be able to run your answer less than $O(n^2)$.
  • Here’s the similar question that I’ve solved before. >> 두 수의 합(A Sum Of Two Numbers)
  • We made its answer run in linear time($O(n)$). You can get an answer that has a quadratic time($O(n^2)$) by applying it as it is.

Climbing Stairs

Posted on 2019-05-15 19:58 | In Problem Solving | Comments:

Problem

https://leetcode.com/problems/climbing-stairs/

Solution

  • d[i] = The number of ways to be i
  • d[i] = d[i-1] + d[i-2]
    If you are standing on the i-1, you only need one step up.
    and If i-2, you only need two step up.

Complexity

  • Time Complexity: $O(n)$
  • Space Complexity: $O(n)$
