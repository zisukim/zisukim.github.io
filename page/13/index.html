SPF(Sender Policy Framework)

Posted on 2019-03-09 15:09 | In ETC | Comments:

What is SPF?

The SPF is an email authentication technique which prevent spammers from sending messages.
It allows the recipient to confirm whether the sender infomation matches the actual mail server.
For this check, A sender has to add its information in the DNS in advance.

  • Sender: Register an SPF recored that represents your mail server and policy in the DNS.
  • Recipient: Check the SPF record registered in the sender’s DNS when receiving e-mail.
    After comparing it in the record and the outgoing IP in the e-mail, decide whether to receive it or not.

SPF Record

A SPF record is published in the DNS and it contains a list of authorized email servers which can send email on behalf of your domain name.
If a sender isn’t in the record section, It may be considered illegitimate and can be rejected by the receiver.

TXT format

Description
v=spf1 The version of SPF it is using
include It continas records to allow for sending to the domain.
all 1. -: Fail. Do not accept mail sent from anywhere but the above list
2. ~: Soft Fail. Allow mail whether or not it matches the parameters in the record
3. ?: Neutral

Merge intervals

Posted on 2019-02-27 08:15 | In Problem Solving | Comments:

Problem

https://leetcode.com/problems/merge-intervals/

Solution

  • [1, 2] and [3, 4] can not be merged. You can merge overlapping parts, not consecutive numbers.
  • If end in privious interval is greater than start in next interval, It can be regarded as overlapping.

Complexity

  • Time complexity: $O(nlogn)$
  • Space complexity: $O(1)$

Reverse Numbers

Posted on 2019-02-23 07:37 | In Problem Solving | Comments:

Problem

https://leetcode.com/problems/reverse-integer

  1. If input is negative, the reversed numberse is also negative.
  2. If input is out of Integer ranges, it returns 0.

Solution

Pop digits and Push them

  • To reverse numbers, we just pop last digit off and push it to the back of new values.
  • We would use mod to get units digit.
  • We have to check input is negative number or not.
  • Because we use mod to reverse input, we record input is negative or not. then make it positive number.
  • At last, We check this solution would cause an overflow.

Complexity

  • Time complexity : $O(logx)$
  • Space complexity : $O(1)$
