Redundant Conenction

Posted on 2019-06-01 05:13 | In Problem Solving | Comments:

Problem

https://leetcode.com/problems/redundant-connection/

Solution

DFS

  • Detect cycle in this undirected graph.
  • When I defined adj as the largest two-dimensional array with following source, I got a Time Limit Exceeded.
  • If only linked edge information is saved, the action can be reduced.

Union Find

  • The Union-Find is one of the fastest algorithms to find connected components given an edge list.

Complexity

DFS

  • Time Complexity: $O(N^2)$
  • Space Complexity: $O(N)$

Union Find

  • Time Complexity: $O(N)$
  • Space Complexity: $O(N)$

Climbing Stairs

Posted on 2019-05-15 19:58 | In Problem Solving | Comments:

Problem

https://leetcode.com/problems/climbing-stairs/

Solution

  • d[i] = The number of ways to be i
  • d[i] = d[i-1] + d[i-2]
    If you are standing on the i-1, you only need one step up.
    and If i-2, you only need two step up.

Complexity

  • Time Complexity: $O(n)$
  • Space Complexity: $O(n)$
