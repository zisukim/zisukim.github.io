PyCharm에서 PyQt5 설치하기

  1. PyCharm을 실행시킵니다.

  2. 메뉴의 File - Settings에 들어갑니다.

  3. 좌측 메뉴에서 Project: {프로젝트이름} - Project Interpreter를 들어갑니다.

  4. Setting창에서 Project Interpreter가 우리가 함께 설치한 3.6.3 버전인지를 확인합니다.

  5. 아래에는 설치한 패키지 목록들이 나오는데 우측에 + 버튼을 눌러 추가를 해보도록 합시다.

  6. 위 검색창에 pyqt5라고 검색을 하고, “PyQt5” 패키지를 설치해줍니다.

  7. PyQt5의 설치가 완료되었습니다 ! python 파일에 import PyQt5를 선언하고, 더이상 빨간 줄이 뜨지 않는 것을 확인합니다.

PyCharm에서 PyQt5 설치하기 (Mac OS)

Posted on 2017-11-12 10:44 | In Python | Comments:

PyCharm에서 PyQt5 설치하기

  1. PyCharm을 실행시킵니다.

  2. 메뉴의 Preferences에 들어갑니다.

  3. 좌측 메뉴에서 Project: {프로젝트 이름} - Project Interpreter를 클릭합니다.

  4. Project interpreter가 우리가 함께 설치한 3.6.3 버전인지 확인합니다.

  5. 아래 + 버튼을 클릭하면 패키지를 추가할 수 있는 창이 뜹니다.

  6. pyqt5를 검색한 후 설치합니다.

  7. PyQt5가 설치되었습니다 ! 패키지 목록에 추가된 것을 확인합니다.

  8. python 파일을 열어 import PyQt5를 수행시켜봅니다. 더이상 빨간 줄이 뜨지 않는 것을 확인할 수 있습니다.
