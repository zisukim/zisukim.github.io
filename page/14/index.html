Two Sum

Posted on 2019-01-03 13:30 | In Problem Solving | Comments:

Problem

https://leetcode.com/problems/two-sum/

Solution

  • You may assume that each input would have exactly one solution.
  • You return indices of two numbers.
  1. Brute force
  • The simplest solution is just comparing every possible pairs. The time complexity could be $O(N^2)$. Because this problem doesn‚Äôt use same element of them, it will be $N * (N-1)$.
  1. Using dictionary
  • We traverse the list containing $N$ elements twice.
  • A dictionary is built for looking up values in constant time.
  • In the first iteration, we add the value of elements as key and the index as value on dictionary.

  • Then, in the second iteration, we check if there is the number to be sum.

  • And‚Ä¶ One loop is enough to get an answer if we care the order of outputs.

Complexity

  • Time Complexity: $O(N)$
  • Space Complexity: $O(N)$

Subnet Mask

Posted on 2018-11-03 03:23 | In ETC | Comments:

Subnet (Sub network)

  • A subnet is a locgical subdivision of an IP network.
  • The main purpose of subnetting is to help relieve network congestion and improve network performance.

Subnet Mask

  • A subnet mask is a number that defines a range of IP addresses available within a network.

Calcurate IPv4 Sub Mask

1. Conver to binary

IP(Decimal) 192.168.63.0
IP(Binary) 10101100.10101000.00111111.00000000
Subnet Mask(Binary) 11111111.11111111.11111111.00000000
Subnet Mask(Decimal) 255.255.255.0

2. Calcurate the subnet address
Perform a bit-wise AND operation.

IP 10101100.10101000.00111111.00000000
Subnet Mask 11111111.11111111.11111111.00000000
Result 10101100.10101000.00111111.00000000

3. Caculate total number of subnets and hosts per subnet

(10101100.10101000.00111111).00000000

  • Subnet count: $2^0$ = 1
  • Subnet address: 192.168.63.0
  • Broadcast address: 192.168.63.255
  • Host count: $2^8$ = 256 => 254 (256 - 2)
  • Range: 192.168.63.1 ~ 192.168.63.254
