FreeMarker는 무엇인가

  • FreeMarker Page
  • 템플릿 엔진. 자바 라이브러리
  • 템플릿에 변경하는 데이터를 적용하여 결과물을 만들어준다.
  • 템플릿은 FreeMarker Template Language (FTL)로 작성한다.
    • 간단하고, 지원하는 기능이 많다. 내가 사용할 때 필요한 기능은 모두 지원하더라. (이런 것도 있나? 하면 있는..)

FreeMarker를 사용하여 문자열을 처리해보자

  • FTL로 작성된 String에 변수를 정의한 map 형태의 데이터를 적용해보자.
  • 현재 지원하는 최신 버전은 2.3.28 이다.
    • 버전에 따라 지원하지 않는 문법들이 있다. (null check 같은 경우 2.3.7 이후 문법이 바뀌었다.)
