State과 Binding - SwiftUI

Posted on 2019-06-30 09:27 | In App , SwiftUI | Comments:

Docs

  • Documentation > SwiftUI > State and Data flow > State
  • Documentation > SwiftUI > State and Data flow > Binding

State

  • 🙋사용자 -(버튼을 클릭하거나 입력을 하거나) -> Action -(값이 변경)-> State -(업데이트) -> View
  • State 값이 변경되면 View는 무효화되고 다시 body가 계산되어 보여집니다. (rerendering)

Examples

Binding

  • Binding을 사용하면 View와 View Model간의 양방향 연결을 만들 수 있습니다.
    View ↔ View Model
  • 연결을 맺을 property에 Binding을 선언하고, $연산자를 사용하여 정의합니다.
  • 좀 더 복잡한 구성(여러 property가 있거나 여러 뷰에서 공유해야하는 property가 있는 경우)인 경우 @ObjectBinding을 대신 사용하면 좋습니다.
