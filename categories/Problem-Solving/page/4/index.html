Max Consecutive Ones II

Posted on 2019-04-27 18:31 | In Problem Solving | Comments:

Problem

https://leetcode.com/problems/max-consecutive-ones-ii/

Solution

  • The only one difference with Max Consecutive Ones is that you can flip at most one 0.
  • If you flip the 0 which is 2nd index, the prevCount would store the number of 1s before the 2nd index including itself.
  • The count doesn’t care about flipping.
  • The candidate of answer is sum of the count before and after the filp.

Complexity

  • Time Complexity: $O(n)$
  • Space Complexity: $O(1)$

Spiral Matrix

Posted on 2019-04-18 05:30 | In Problem Solving | Comments:

Problem

https://leetcode.com/problems/spiral-matrix/

Solution

  • Describe it as people ususally think.
  • You would change your direction when you meet the edge of each side or the point that you already have seen.
  • Turning around is a repeating pattern. In case of y as row, It updates 0(stay), 1(down), 0(stay), -1(up) and repeats this.
  • It repeats loop for m * n times so that it can access all points of the matrix that is requested.

Complexity

  • Time Complexity: $O(N)$ N is number of elements in matrix.
  • Space Complexity: $O(N)$
