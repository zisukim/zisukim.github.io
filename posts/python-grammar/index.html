append()

  • It appends an element to the end of the list.

pop()

  • It removes the last element and returns value.

replace()

  • The method replace() returns a copy of string which has been replaced with new.
  • max: A number specifying how many occurences of the old value you want to replace. Default is all occurences.

sort(key=, reverse)

  • It sorts elements of a given list in acending or descending order.
  • You can also use Python built-in function sorted() to sort a list.

Built-in Functions

all(x)

  • An input argument can be iterable data types like lists, tuples and dictionaries.
  • If you assume that the input argument is x..
  • The method returns True if all elements of x are true, False if any elements of x are false.

int(x)

  • It returns a string or a real number in an integer.

divmod(x, y)

  • It takes two numbers and returns a tuple consisting of quotient and remainder.
  • If x and y are integers, the return value is same as (x // y, x % y).

ETC

itertools.product(*iterables[, repeat])

  • Roughly equivalent to nested for-loops in a generator expression.
    For example, product(A, B) returns the same as ((x,y) for x in A for y in B).

Unpacking Argument Lists

  • When arguments are already list or tuple, they need to be unpacked for function call requiring separate positional arguments.
  • You write function call with * operator to unpack a list or tuple.
  • The dictionaries need to be unpacked with ** operator.
