Read Large Excel File 2. XLS

Posted on 2019-06-08 13:30 | In Web , Java | Comments:

Why

  • XSSF and SAX doesn’t support the XLS files. We would use the other API for the HSSF.
  • What are the XSSF and HSSF?

Apache POI Event API

  • Apache POI Event API(HSSF Only)

Add the dependency (maven)

Implements a Listener

  • The process of the header and using a user-defined consumer is skipped. It is the same as the XLSX Sheet Handler.
  • We have to implement an interface HSSFListener. The processRecord() listens for incoming records and handles them as required.
  • The work is depending on the type of record. It’s a little annoying compared to the XSLX Content Handler.

Parse the XLS files

  • The MissingRecordAwareHSSFListener triggers our listener for all rows and cells, even the ones that aren’t actually stored in the file.

Notes

The number of rows in HSSFSheet is limited.

I will post How to parse the CSV files next time.
